# Ebytr
Neste repositório será desenvolvido uma aplicacão para o desafio da blitz de carreira efetuada na trybe com o intuito de simular um desafio técnico real.

## Como usar
Rode esses comandos em seu terminal para clonar o projeto.

```
# Clonando
$ git clone git@github.com:thiago-gmacedo/ebyrt-challenge.git

# Entre na pasta
$ cd ebyrt-challenge

# Removendo a origem do repositório
$ git remote remove origin
```

### Instalando dependências do backend e rodando o servidor
```
# Entre na pasta server
$ cd server

# Instale as dependências
$ npm i

# Inicie o servidor de desenvolvimento
$ node index.js
```

### Instalando dependências do frontend e rodando a aplicação
```
# Entre na pasta client
$ cd client

# Instale as dependências
$ npm i

# Inicie a aplicação
$ npm start
```
